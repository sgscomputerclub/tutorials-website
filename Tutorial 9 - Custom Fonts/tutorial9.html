There are more fonts available than the ones installed on your computer right now. You can download more fonts from websites such as dafont.com and fontsquirrel.com.

IN THE CSS, I HAVE MADE IT SO THAT "H2" TAGS DISPLAY IN A CUSTOM FONT THAT I CHOSE.

The convension is to use a woff file to display custom fonts on webpages. You can convert tff files to woff files here.

Most browsers will allow you to use a tff file (the most common format) to use as a custom font, but some browsers Internet Explorer won't allow this (only woff files). This is the main reason why many people begun to hate Internet Explorer - in fact some old versions of Internet Explorer (below 7) don't even support custom fonts (font faces) at all!
